Run the Rhythm Starter locally

You're seeing the GitHub Pages placeholder for NotTapTap. The playable rhythm prototype needs a live backend for Spotify and Apple Music authentication, which GitHub Pages cannot host.

To experience the app, clone the repository and follow the setup steps in the README: 

pnpm install
pnpm dev

If you want a static Pages build, deploy the Express server elsewhere, run GITHUB_PAGES=1 pnpm --filter client build, and publish the generated files from client/dist into this docs/ folder.
