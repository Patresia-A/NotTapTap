Run the Rhythm Starter locally

This repository uses a Vite + React client and a Node/Express backend that require Spotify and Apple Music credentials. The interactive app is not bundled into GitHub Pages because the backend cannot run here.

Want to host the UI on Pages anyway? Build the client with GITHUB_PAGES=1 pnpm build inside the client/ folder, publish the contents of client/dist, and point it at a separately hosted copy of the backend.

For local development, clone the repository and follow the setup steps in the README: 

pnpm install
pnpm dev

Once both servers are running, open http://localhost:5173 in your browser to connect your music service and play the demo chart.
